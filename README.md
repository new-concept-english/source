# source

新概念资源文件，后续你可以基于这个仓库来复用。

目前仓库的所有音频（美音、英音）是齐全的，但是课文和插图的资源还正在处理中。

> 如果你有 chatGpt 的会员能提供使用，一定可以节省很多效率。

## 使用方法

```sh
npm i @boses/source
```

使用方式

```js
import volume1 from '@boses/source/1.json';
// or
import { volume1, volume2, volume3, volume4 } from '@boses/source';
```
